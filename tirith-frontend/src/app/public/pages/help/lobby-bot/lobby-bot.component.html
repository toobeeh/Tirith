Server Home & Lobby Bot



Bring your skribbl community together on discord



How it works

When you have a discord server, you can set it up as a typo server home.
This allows other people on your discord server to connect to this server - no one else can connect, so it's private!

On a home server, you have access to a dedicated Palantir Bot.
This Bot will check frequently who of your server members (with a typo account) are online on skribbl.
If someone is online, it shows their lobby with stats and the lobby link in a discord channel.
Everyone on the server can quickly join using the lobby link, or with shortcut buttons directly on skribbl.io!

Additionally, you can also see a leaderboard of all connected people, or set up a discord channel to post images directly from skribbl.

Set Up Server Home

To use a server as server home, a typo member with an active patreon subscription has to support the server.
This can be done with the command /patron home anywhere on the server.
Patrons can select only one server at a time. If it's needed, a different server can be selected after a cooldown of seven days.
It is possible that more than one member supports a server as server home, but it won't bring additional perks.

As soon as the server is supported by a patreon subscriber, a server admin can invite the Palantir Lobbies Bot.
The invite link for the bot is shown when the "/patron home" command is used.
The bot is exclusively for this server and will stay as long as the server is supported by a patreon member.

The last step is to choose a channel on your server where the Lobby Bot will show all lobbies. This has to be done by a server admin.
The bot creates a message and edits it every 20 seconds. It is recommended to create a new channel, where only the bot can send messages.
With the command /server lobbies <#channel> the bot will start right away to show lobbies!

Customize the Server Home

When the setup is completed, you can do a few more things to customize your typo server home.

The Lobby Bot supports the same commands as the public Palantir bot, but for this one you can choose a custom Prefix.
To set a prefix, a server admin can use the command /server prefix <prefix>

You can also add an Image Post channel with the command /imagepost add <#channel> <name>.

With the command /server info you can always see the details of your server home.

Lobbies Channel

The Lobbies Channel is the heart of your typo server home.
It displays all lobbies of connected members and lets others quickly join either by the link or via a join button on skribbl.io.

To be displayed there, members must connect to the server home first - this can be done using the invite link in the Lobbies Channel.

Server Leaderboard

As soon as the server home is set up, you can use the /leaderboard bubbles command on this server to see a ranking of all connected members, ordered by their bubble amount.
If you want to rank by drop score, you can use the command /leaderboard drops.

Image Post Channel

When a server admin has set up an Image Post channel on the server, connected members can quickly send images from skribbl to a server channel.

To send an image, you can then click the letter-icon on skribbl and choose the name of the channel where you want the image to be posted.

Server admins can add an image post channel with the command /imagepost add <#channel> <name>.
The channel option sets the server channel where posted images will appear, and the name option sets the name which users will see on skribbl.
You can see all image post channels of the server with /imagepost list or remove one with /imagepost remove <name>.
