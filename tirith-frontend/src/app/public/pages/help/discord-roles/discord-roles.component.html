Linked Roles



Show Palantir stats on your Discord profile



How it works

When you see this page, you have probably just set up linked roles. Congrats!
  • You will now have a profile card on the server profile where you connected.
  • Whenever you are online on Skribbl, your stats on your profile will refresh automatically.
  • If the server has set it up, you can unlock special server roles with your Palantir stats.
  • If you want to remove the connection, go to your Discord settings -> Athorized Apps, and deauthorize Palantir.
